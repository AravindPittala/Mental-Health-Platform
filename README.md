# Description
A web-based mental health support platform designed to provide accessible, user-friendly emotional support and self-help resources.
Focused on improving mental well-being through guided assistance, awareness features, and a clean, responsive interface.
#Features
AI-powered mental health self-assessment
Personalized feedback and actionable suggestions for users
Focus on early awareness and self-reflection for mental well-being
Simple, responsive, and user-friendly web interface 
#Tech Stack
Frontend: HTML, CSS, JavaScript  
Backend: Python (Flask)  
Database: SQLite
